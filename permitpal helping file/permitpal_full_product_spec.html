The problem — regulations just got brutal

US: 50-state patchwork
Every city has different permit types, renewal dates, night caps, and tax obligations. NYC banned most unhosted STRs. Austin requires license numbers in every ad. Hosts operating across cities are drowning in paperwork.
EU: Regulation 2024/1028 just kicked in
From May 2026, every EU host must register and display a registration number on all listings. Platforms remove non-compliant listings automatically. Fines of €10K–€60K for violations. Spain fined Airbnb €64M for unlicensed listings.
Australia: three-tier chaos
NSW: register + 180-night cap. Victoria: 7.5% Short Stay Levy + strata bans. WA: mandatory registration from Jan 2025. Brisbane: new permit laws mid-2026 with $140K fines. Every state different. Every council adds its own rules on top.
No one is tracking this for hosts
Hosts manage permits in spreadsheets, miss renewal dates, exceed night caps, and get delisted or fined. There is no modern, affordable compliance tool built specifically for individual Airbnb hosts and small operators.

Target customers

Solo Airbnb host (1–3 properties)
Managing one home or a couple of investment properties. Not a professional. Gets fined because they didn't know about a permit renewal. Primary ICP.
Small portfolio operator (4–20 properties)
Managing across multiple cities or countries. Uses co-hosts or a VA. Needs multi-property compliance dashboard. Willing to pay $79–$149/mo.
Property manager (managing others' STRs)
Manages compliance on behalf of clients. Needs white-label reporting. Team seats. This is the highest-LTV segment.

Core compliance engine

All markets
Property compliance profile
Enter property address → PermitPal identifies applicable jurisdiction, fetches current rules, returns: what permits you need, current night cap, required registration numbers, and applicable taxes.
All markets
Permit & licence tracker
Store all permits with issue date, expiry date, permit number, issuing authority. 90/30/7 day renewal alerts via email + SMS. One-click link to the renewal portal for 200+ jurisdictions.
All markets
Night-cap counter
Sync with Airbnb/VRBO/Booking.com via iCal or API. Counts booked nights per calendar year per property. Alerts at 80%, 90%, 100% of annual limit. Blocks bookings automatically when cap is reached (API hosts).
All markets
Registration number manager
Store registration numbers per listing per platform. One-click copy to update Airbnb listing description. Validate EU registration numbers against national registries (May 2026 requirement).
All markets
Compliance score dashboard
Every property gets a live compliance score 0–100. Red = action needed. Amber = renewal due soon. Green = fully compliant. Portfolio view for multi-property hosts.
All markets
Document vault
Upload and store permits, safety certificates, insurance docs, fire inspection reports. Auto-tagged by expiry date. Shared access for property managers and accountants.

US-specific features

US only
City rule database (500+ cities)
Live database of STR rules for 500+ US cities — permit types, night caps, zoning restrictions, primary residency requirements, renewal cadence. Updated monthly via regulatory monitoring.
US only
TOT / occupancy tax tracker
Track Transient Occupancy Tax rates by city + county. Flag direct bookings requiring self-remittance (Airbnb remits state tax, not always local). Export quarterly tax summary for accountant.
US only
Zoning check
Enter property address → check against zoning database to confirm STR is permitted in that zone. Warning if property is in restricted residential zone before host invests in a permit.
US only
License display checker
Scan Airbnb listing URL → verify permit/license number appears in listing description. Alerts if Austin, NYC, or other mandate-cities detect a missing number (causes automatic delisting).

EU-specific features (biggest differentiator)

EU only
EU Reg 2024/1028 compliance wizard
Step-by-step registration guide for each EU member state — France (Déclaration), Spain (NRU), Netherlands, Germany. Links directly to each national registration portal. Tracks application status.
EU only
Registration number validator
From May 2026, platforms auto-remove listings with invalid reg numbers. PermitPal validates numbers against country registries and alerts immediately if a number becomes invalid or suspended.
EU only
Night-cap monitoring (EU cities)
Paris: 120 nights. Amsterdam: 30 nights (proposals for 15). Lyon: 120 nights. Barcelona: approaching phase-out. Track against each city's current cap. Alert before breach — platforms now auto-verify this data monthly.
EU only
France Loi Le Meur compliance
Track principal residence status (275 days/yr occupancy requirement). Energy rating alerts (DPE requirement from 2025). Mayor-set night reduction monitoring for each French commune.
EU only
Platform data reconciliation
EU platforms now share host data monthly with authorities. PermitPal shows what data has been reported about your listing — so hosts can cross-check for discrepancies before an authority does.

Launch sequence

Month 1–2
Build EU compliance wizard first

EU Reg 2024/1028 fully applies May 2026 — every EU host needs to register. This is a time-sensitive, high-urgency pain that gives you a natural deadline to sell against. Build the registration wizard for France, Spain, Netherlands, Germany first. Target EU hosts immediately.

Month 2–4
AU beta — NSW + Victoria + WA

Three states with the most complex requirements in 2025. NSW STRA register + 180-night cap, Victoria 7.5% levy self-reporting, WA new mandatory registration. Every AU host with a property in these states has an active pain right now.

Month 3–6
US — top 10 cities

NYC, SF, Austin, Nashville, Miami, Denver, Seattle, Chicago, LA, Honolulu. Build the city rules database for these 10 cities first. They cover 80% of US host demand. Expand to 500 cities by month 12 via a regulatory data team or partnership.

Month 6–12
SEO content moat

Write the definitive guide to STR compliance for every target city. "Austin Airbnb permit requirements 2025", "Paris short-term rental registration number", "Sydney 180-night cap tracking". Zero competitors doing this well. Organic traffic becomes the primary acquisition channel.

Month 9+
Property manager channel

Partner with PMS tools: Hospitable, Hostaway, Guesty. Build integrations that pull booking data automatically for night-cap tracking. Property managers become the distribution channel — they manage 20+ properties each and will pay Operator plan.
